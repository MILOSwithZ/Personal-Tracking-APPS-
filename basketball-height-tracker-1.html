⚙️ Setup Google Drive Sync

Masukkan Google OAuth Client ID untuk mengaktifkan cloud sync.

Cara dapat Client ID (±5 menit):
1. Buka console.cloud.google.com
2. New Project → APIs & Services → Enable APIs → aktifkan Google Drive API
3. OAuth consent screen → External → isi nama app
4. Credentials → Create credentials → OAuth client ID → Web application
5. Authorized JS origins = URL tempat kamu host file ini
6. Copy Client ID dan paste di bawah
